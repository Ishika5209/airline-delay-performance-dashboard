# ✈️ Airline Delay & Performance Dashboard

An interactive Power BI dashboard designed to analyze airline performance by examining delays, cancellations, and on-time trends.
This project transforms raw flight data into actionable insights using dynamic visualizations and user-driven filters.

## 📊 Overview

The dashboard provides a comprehensive view of airline operations, enabling users to explore patterns in delays, evaluate airline efficiency, and identify factors affecting performance.

## 🔍 Key Insights

* On-time performance is approximately 60%, indicating moderate operational efficiency
* Average arrival delay is around 7.6 minutes across flights
* Certain airlines exhibit consistently higher delay patterns
* Cancellation rate (~3.8%) impacts overall reliability

## ⚙️ Features

* KPI Cards (Total Flights, On-Time %, Avg Delay, Cancellation Rate)
* Monthly delay trend analysis
* Airline-wise delay comparison
* Interactive slicers (Airline, Month, Origin Airport, Day of Week)
* Visuals: Line Chart, Bar Chart, Donut Chart, Gauge

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Microsoft Excel

## 📁 Dataset

Flight performance dataset containing information on airlines, delays, cancellations, and scheduling metrics.

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use slicers to interact and explore insights

## 📌 Project Objective

To build an intuitive and interactive dashboard that helps in understanding airline performance and supports data-driven decision making.
